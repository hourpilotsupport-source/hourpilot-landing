# # HourPilot

Lightweight weekly planning and workload insights for Toggl Track.

HourPilot adds:
- weekly project planning
- planned vs actual tracking
- workload capacity visibility

directly inside Toggl.

## Features

- Plan weekly hours per project
- Compare planned vs actual tracked time
- Visualize workload capacity
- Local-first
- No backend
- No analytics

## Privacy

HourPilot stores data locally in Chrome storage.
No personal data is collected or transmitted to external servers.
